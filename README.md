# Java Coding Practice

Simple Java coding problems for practice.

## Prerequisites

- Java JDK installed (Java 17 or higher recommended)
- A text editor or IDE (e.g. IntelliJ IDEA or VS Code)

## How to run

Compile the file:

```bash
javac {file-name}.java
```

Run the compiled class:

```bash
java {file-name}
```
